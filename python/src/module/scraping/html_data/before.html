お問い合わせ

ご不明点やご要望がございましたら、お気軽にお問合せください。ただし、お問い合わせいただく前に、以下の注意事項をご確認いただきますようお願いいたします。

規約

  • お問い合わせする際には、下記項目を必須項目としています。
    • 氏名
    • メールアドレス
    • 問い合わせ事項
  • お問い合わせいただいた内容には、原則として3営業日以内に返信いたします。ただし、お問い合わせの内容や状況によっては、返信に時間がかかる場合がございますので、予めご了承ください。
  • 個人情報の取り扱いに関しましては、 有留大学個人情報取り扱いをご参照ください。
  • お問い合わせ後、自動返信メールをお送りします。もし自動返信メールが届かない場合は、迷惑メールフォルダを確認いただくか、入力いただいたメールアドレスが正しいかご確認ください。
